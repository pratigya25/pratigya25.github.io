Proxemics to a Virtual Agent in VR

December 2024 - Present | Media Project
Unity Hub 2022.3.59f1 C# Blender Character Creator Mixamo Convai VR

Project Overview

This research project evaluates the role of proxemics (personal space and distance) in personal versus general communication tasks with virtual agents in Virtual Reality. The study aims to understand how users naturally interact with AI-powered virtual characters in immersive environments.

Key Features

  • Immersive VR environment set in a traditional pirate-themed bar
  • AI-powered virtual agent with adaptive conversation capabilities using Convai
  • Real-time proxemics data logging to measure user comfort zones
  • User study framework to gather comprehensive interaction data
  • Analysis of spatial behavior patterns in VR social interactions

Research Goals

  • Measure comfortable interaction distances in VR social scenarios
  • Compare proxemics behavior between personal and general tasks
  • Inform design guidelines for engaging VR social experiences
  • Contribute to understanding of human-AI interaction in immersive spaces

Technical Implementation

The project leverages Unity's VR capabilities combined with Convai's AI conversation system to create realistic agent interactions. Character models were created using Character Creator and animated with Mixamo, while the environment was crafted in Blender. The system logs detailed spatial data including user position, orientation, and distance from the agent throughout interactions.

Ecoverse - 3D Educational Game

November 2024 - February 2025 | Game Development Project
Unity Hub 3.10.0 C# Blender Google Speech-To-Text Gamification

Game Concept

Ecoverse is an engaging 3D educational game that promotes sustainability through interactive gameplay. Players take on the role of a stranded astronaut who must learn about sustainable practices while collecting fuel components to escape a planet that has been polluted with garbage by previous space travelers.

Level Design

  • Level 1: Solar Collection - Navigate the environment to collect solar panels while dodging garbage balls. Earn sustainability knowledge points through exploration and environmental storytelling.
  • Level 2: Knowledge Challenge - Answer quiz questions based on earned knowledge points to gain protective shields. Tests player understanding of sustainability concepts.
  • Level 3: Ice River Escape - Traverse a melting ice river showcasing climate change effects. Use earned shields for protection while collecting wind energy components to power the spaceship.

Innovative Features

  • Voice control mechanics using Google Speech-To-Text API
  • Progressive difficulty scaling based on player performance
  • Educational content seamlessly integrated into gameplay
  • Environmental storytelling through level design
  • Gamified learning with immediate feedback and rewards

CGI Speed Perception Study

May 2024 - July 2024 | User Research Project
Unreal Engine 5.3.2 Python Quantitative Analysis CGI UX Research

Research Overview

This study investigates human perception of speed in computer-generated environments, specifically designed to simulate an autonomous vehicle's windshield view. The research aims to understand how accurately people perceive vehicle speed through CGI simulations, which has critical implications for future autonomous vehicle interface design.

Methodology

  • Designed photorealistic CGI scenes in Unreal Engine 5 simulating various driving scenarios
  • Conducted controlled user studies with multiple participants
  • Gathered quantitative data on speed perception accuracy across different scenarios
  • Analyzed consistency of perception across demographic groups
  • Compared CGI perception vs real-world speed estimation

Research Findings

The study provided valuable insights into how users perceive speed in virtual environments, revealing patterns in perception accuracy and identifying factors that influence speed estimation. These findings directly inform better user experience design for autonomous vehicle interfaces and can help create more intuitive visual feedback systems.
